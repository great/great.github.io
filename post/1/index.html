Serve *.github.io using hugo

February 10, 2016

github.io를 시작해 보려고 어떻게 하는 것인지 찾아 보니 보통 Jekyll이라는 것을 가지고 만드는 것 같다. 그런 줄만 알고 그러려니 했는데 Jekyll 자체를 구동하기 위해 ruby기반으로 이것저것 설치를 해야 해서 고통스럽다. 갑자기 다른 곳에서 과연 그럴 일이… 뭔가 쓰고 싶거나 할 때 ruby부터 설치하고 어쩌고 해야 하는 과정이 매우 번거로워 보인다.

그러던 중 go를 가지고 만든 hugo를 알게 되었고 go 특성(?)상 번거로운 실행 환경 구성 문제가 없어 날로 먹을 수 있다 마음에 든다.

대부분 환경에서 구동 가능한 hugo 바이너리를 간단히 받을 수 있으니 “특별한 경우”에 글 하나 올리겠다고 ruby 환경 구성하고 Jekyll 설치까지 해야 하는 그런 번거로운 일은 안 해도 된다.
